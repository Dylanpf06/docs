---
title: "Ds Cloud Services Status"
sidebarTitle: "/v2/service/status"
description: "Check the real-time status of Ds Cloud services"
api: "GET https://api.dscloud.app/v2/service/status"
authMethod: "none"
---

<Info>
  This route has a 5-minute cache and does not reflect the real-time status of
  status.dscloud.app.
</Info>

### Response

<ResponseField name="status" type="string">
The current status of Ds Cloud, with `online` indicating the service is online, `degraded` indicating issues, and `unknown` indicating unavailable.
</ResponseField>
<ResponseField name="message" type="string">
Message about the current status of Ds Cloud services.
</ResponseField>
<ResponseExample>
```json
{
    "status": "online",
    "message": "All services are online."
}
```
</ResponseExample>
