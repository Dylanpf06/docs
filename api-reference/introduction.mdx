To interact with the **Ds Cloud API**, you need an API key. This key serves as your unique identifier and allows you to authenticate requests securely. Follow the steps below to retrieve and use your API key.

## Obtaining Your API Key

1. Navigate to the [Developer Dashboard](https://dscloud.com/dashboard/settings).

2. Locate the **API Key** section under **Settings**.

3. Generate or copy your existing API key.

***

## Using Your API Key

To authenticate with the Ds Cloud API, include your API key in the `Authorization` header of your HTTP requests. Below is an example of how to format the header:

```bash
'Authorization': '<api_key>'
```
